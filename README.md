## 🚀 Overview
Shadow Realm Rev IP is an advanced domain, IP, and subdomain harvester. Given a target IP or domain, it performs reverse IP lookups to discover all sites hosted on the same server, resolves subdomains, and checks HTTP status for each discovered host using concurrent lookups.

---

## 🎮 Features

### 🔎 Recon Features
- Reverse IP domain discovery
- Subdomain resolution
- HTTP status code checking
- Concurrent multi-target scanning
- Proxy support for anonymity

### ⚙️ System Features
- ⚡ Concurrent futures multi-threaded
- 📡 Proxy and retry adapter
- 📊 Live discovery counter
- 📁 Results saved to output files
- 🌑 Colorized terminal output

## 📦 Installation

```bash
1. Download the ZIP file
2. Extract the archive
3. Install dependencies:
   pip install -r "Shadow Realm RevIP requirements.txt"
4. Run the script:
   python "Shadow Realm RevIP.py" -t <target>
```
